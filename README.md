# Agentic Platformer

A small Godot 4.6 Standard/GDScript prototype for an AI-assisted 2.5D platformer.

## Run

Open `game/project.godot` in Godot, then press Play.

Current controls:

- `Start` begins or resumes the prototype.
- `Exit` asks for confirmation and closes the game.
- `Escape` opens the menu while in game.
- `A` / `Left Arrow` moves left.
- `D` / `Right Arrow` moves right.
- `Space` jumps.
